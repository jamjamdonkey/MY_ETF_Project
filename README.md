# 📊 DONKEY ETF Project & Survival Racing MVP
> **"어려운 퀀트 데이터를 대중의 언어로 번역하는 금융 콘텐츠 파이프라인"**

본 레포지토리는 파이썬 기반의 심도 있는 자산배분 퀀트 리서치인 **[본진: DONKEY ETF]**와, 해당 리서치 결과를 일반 대중(20대 대학생)이 쉽게 체험할 수 있도록 개발한 웹 기반 인터랙티브 콘텐츠인 **[파생 프로덕트: 1천만 원 생존 레이싱]**을 함께 담고 있습니다.

## 🔍 상세 소개 (Details)

### 1. 본진: DONKEY ETF 리서치 
시장의 메가 트렌드(모멘텀)와 하방 경직성(퀄리티, 배당)을 동시에 잡기 위해 기획된 다이렉트 인덱싱(Direct Indexing) 포트폴리오입니다.
* **주요 기능:** 29개 핵심 종목의 상관계수 분석 및 자산 배분 비중 산출
* **데이터 검증:** 샤프(Sharpe), 소르티노(Sortino), 오메가(Omega) 지수 등 심화 퀀트 지표를 통한 성과 평가
* **팩터 분석:** SPY, SCHD, QQQ 등 시중 대장 ETF와의 6-Factor 노출도(Exposure) 비교 시각화

### 2. 파생: 1천만 원 생존 레이싱 (Web MVP)
본진에서 얻은 '잃지 않는 투자'의 철학을 20대 대학생 오프라인 마케팅에 적용하기 위해 개발한 최소기능제품(MVP)입니다.
* **🎮 [웹 게임 바로가기]**: `https://(강한님의 깃허브 아이디).github.io/(레포지토리 이름)/` *(여기에 아까 만든 깃허브 페이지 링크를 넣으세요!)*
* **작동 원리:** `fetch_prices.py`를 통해 추출한 과거 5개년 실제 주가 데이터를 기반으로 구동됩니다.
* **핵심 로직:**
  1. 참가자가 10개 자산의 비중을 직접 조절 (슬라이더 UI)
  2. Canvas API를 활용해 과거 폭락장(-MDD)과 상승장 구간을 60초간 경주마처럼 레이싱 애니메이션으로 구현
  3. 게임 종료 시 **'수익률/변동성(안정성 점수)'** 기반으로 순위 산정 및 개인 성향에 맞는 **ACE ETF 자동 매핑(Mapping)**

## 🎯 시사점 (Lessons Learned)
단순히 수식을 증명하는 것을 넘어, **"데이터의 본질(위험 대비 수익률)을 해치지 않으면서 대중이 열광할 수 있는 도파민(게임)으로 번역하는 기획력과 구현력"**을 증명하고자 했습니다. 코드를 짜는 퀀트 개발자의 시야와, 상품을 파는 마케터의 시야를 통합한 프로젝트입니다.
